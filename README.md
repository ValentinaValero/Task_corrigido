# 📝 TaskMaster Pro - Sistema de Gerenciamento de Tarefas

O **TaskMaster Pro** é um projeto front-end desenvolvido com foco em organização, estruturação semântica e boas práticas de desenvolvimento web.

O sistema simula uma aplicação de gerenciamento de tarefas, apresentando uma página institucional explicando o sistema e uma página de contato com validação em JavaScript.

---

## 🚀 Funcionalidades

- Página inicial institucional sobre o sistema
- Layout moderno e responsivo
- Navbar e footer padronizados
- Página de contato funcional
- Validação de formulário com JavaScript
- Feedback visual de erro e sucesso
- Limpeza automática do formulário após envio
- Organização profissional de arquivos em pastas

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** → Estruturação semântica  
- **CSS3** → Estilização, layout e responsividade  
- **JavaScript** → Validação e interatividade no formulário  
- **Git** → Controle de versão
